**Описание проекта**

Проект Mesto - это интерактивная страница, на которой можно изменить данные, добавить фотографии, ставить лайк понравившемуся фото.

**Какие технологии использовали**

* Grid Layout;
* Flex;
* БЭМ-технологии;
* JavaScript.
