**Описание проекта**

Проект Mesto - это интерактивная страница, на которой можно:
* изменить данные профиля;
* добавить фотографию с названием;
* удалить фотографию;
* поставить и убрать лайк на фото.

**Какие технологии использовали**

* Grid Layout;
* Flex;
* БЭМ-технологии;
* JavaScript.
